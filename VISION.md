# Electric Sheep Fork - Vision

**Long-term Product Goal:**
To breathe new life into the classic Electric Sheep distributed computing project. The goal is to modernize the codebase, ensure cross-platform compatibility, and enhance the visual fidelity of evolving fractal flames while preserving the hypnotic, collaborative nature of the original screensaver.

**Design Direction:**
*   **Modernized Infrastructure:** Migrating away from obsolete dependencies to modern equivalents (e.g., updating FFmpeg APIs, modernizing OpenGL context creation).
*   **Robustness:** Improving thread safety, memory management, and error handling across network downloads, video decoding, and rendering.
*   **Enhanced UX:** Providing a seamless, zero-configuration setup for users, while offering deep customization for power users via an accessible settings interface.
